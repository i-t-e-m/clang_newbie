コードの解説

#include <stdio.h>

標準入出力をするために必要なライブラリです。
キーボードからの入力を受け付けたり、画面に何かを出力する場合に必要になります。
基本的に常に書くことになるでしょう。

決しておまじないではありません

int main(int argc, char const *argv[])

プログラムが始まる最初の場所（エントリーポイント）です。プログラムはここから始まります。
main関数がどこにも書かれていない場合プログラムは実行されません。

  • intはmain関数の戻り値の型です。void（空を意味する型）にすることもできますが、基本的にmainの戻り値の型はintにします。

  • mainは関数名です。関数はC言語において、意味のあるプログラムの塊です。C言語にはクラスやメソッドはなく関数だけが存在します。基本的に小さな関数を組み合わせることで様々なプログラムを作成します。

  • int argcはコマンドラインから入力される引数の総個数（プログラム名を含む）が入ります。

    プログラム名を含むところに注意してください

  • char const argv[]は引数の文字列を指すポインタの配列を表しています。constや（ポインタ）については今後説明します。現状はコマンドラインに入力された引数の文字列をchar型の配列で取得していることがわかれば大丈夫です。

printf("Hello C World\n");

フォーマット指定付きの標準出力です。引数のダブルクォーテーションで囲われた文字列を標準出力に出力します。変数の表示などは今後説明します。

return 0;

0を呼び出し元のプログラムに戻します。main関数はOSから呼び出されているためOSに0の値が戻ります。
0以外を戻しても問題はありませんが、習慣的に0が正常、0以外が異常を意味するので理由が無い限りmainでは0を返すべきです。
